Phase 1:
Step 1:
	clean the data by saving the .xlsx file as a .csv then running the provided dataCleaner python script. This will produce a CompanyDataCleaned.csv file
Step 2:
	on your Mac open up a terminal navigate to the directory and install mysql version:
	mysql  Ver 9.5.0 for macos14.8 on x86_64 (Home brew)
	by running the command:
	brew install mysql
Step 3:
	run:
	brew install mysql-shell
	to install mysqlsh version:
	mysqlsh   Ver 9.5.0 for macos15 on x86_64 - for MySQL 9.5.0 (MySQL Community Server (GPL))
Step 4:
	run:
	brew services start mysql
	I do not know if this helps but I ran it so it might be nessisary to start mysql
Step 5:
	run:
	mysqlsh
	to start the mysqlsh shell
	then:
	\sql
	(I do not think the \sql) part doesn't actualy do anything but I ran it so you should to to be safe
Step 6:
	connect to the sql database with:
	\connect root@localhost
	then when it asks for the password hit enter
	then when it asks if you would like to save the password hit enter
	then it will say something about autocompleting the password which might mean you will get locked out if you leave since you do not know the password so do not close the window
Step 7:
	realize steps 3-7 probably just made things worse by potentially locking yourself out so switch to a separate tab and go back to the correct directory to run
Step 8:
	time travel to before step 3 and run:
	mysql -u root -p
	then when it asks for a password hit enter. This will start the mysql shell
	(this might not work if you skipped do not time traveling as the password might be different)
Step 9:
	run 
	CREATE DATABASE OGDataBase;
	to create the database then:
	USE OGDataBase;
	to enter the database
Step 10:
	outside of the shell you just created run sqlTableCreationGeneratorScript.py
	this will print out:
	CREATE TABLE OGDataTable (orderID INT,customerID VARCHAR(255),employeeID INT,orderDate DATETIME,requiredDate DATETIME,shippedDate DATETIME,shipVia INT,Freight FLOAT,productID INT,unitPrice INT,quantity INT,discount INT,companyName VARCHAR(255),contactName VARCHAR(255),contactTitle VARCHAR(255),employees_lastName VARCHAR(255),employees_firstName VARCHAR(255),employees_title VARCHAR(255),productName VARCHAR(255),supplierID INT,categoryID INT,quantityPerUnit VARCHAR(255),unitPrice_1 INT,unitsInStock INT,unitsOnOrder INT,reorderLevel INT,discontinued INT,categoryName VARCHAR(255),suppliers_companyName VARCHAR(255),suppliers_contactName VARCHAR(255),suppliers_contactTitle VARCHAR(255));

	copy that text in to this document then copy it from this document then deleate the hidden character between the ( an the o then copy the modified version and run it in the mysql shell. (The second copy is important as the text has hidden characters which will mess you up)
Step 11:
	run 
	SET GLOBAL local_infile = 1;
	to turn on local file copying
Step 12:
	run:
	LOAD DATA LOCAL INFILE '<local path>CompanyDataCleaned.csv' INTO TABLE OGDataTable FIELDS TERMINATED BY ',' ENCLOSED BY '"' LINES TERMINATED BY '\n' IGNORE 1 ROWS;
	making sure to replace <local path> with the path before the csv file to load the data
Step 13:
	give up at trying to make step 12 work
Step 14:
	run sqlInsertInToTableFromCSV.py
	this will spit out a line so long it will take a minute to past in to the terminal. Past this line in to the mysql shell and it will populate the table
Step 15:
	open up a new terminal window to the directory and run:
	mysqldump -u root -p OGDataBase OGDataTable > OGDataTable.sql
